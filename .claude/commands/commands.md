Development & Build

- /dev - Start the Next.js development server
- /install - Install project dependencies

Git & Version Control

- /commit - Creates git commits using conventional commit format with appropriate emojis



Session Management

- **/project:session-start - Start a new development session**
- **/project:session-update - Update the current development session**
- **/project:session-end - End the current development session**
- **/project:session-list - List all development sessions**
- **/project:session-current - Show the current session status**
- **/project:session-help - Show help for the session management system**
